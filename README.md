# [CCMath](https://crates.io/crates/ccmath)

[![](https://img.shields.io/crates/v/ccmath.svg)](https://crates.io/crates/ccmath)

A simple crate for doing math with complex numbers.

Featuring

- Standard complex functions (absolute value, argument),
- Operator overloading for
  - Addition,
  - Assignment addition,
  - Subtraction,
  - Assignment subtraction,
  - Multiplication,
  - Assignment multiplication,
  - Division,
  - Assignment division,
- Trigonometric functions,
- Inverse trigonometric functions,
- Hyperbolic trigonometric functions,
- Inverse hyperbolic trigonometric functions,
- Formatting for printing complex numbers
